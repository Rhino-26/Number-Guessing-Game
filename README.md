🎯 Number Guessing Game

A fun and interactive Number Guessing Game built using HTML, CSS, and JavaScript. The application generates a random number within a defined range and challenges the user to guess it correctly in as few attempts as possible.

After each guess, the app provides helpful hints such as “too high” or “too low,” guiding the player toward the correct answer. The game also tracks the number of attempts and displays a success message when the correct number is guessed.

Designed with a clean and modern interface, the game offers a smooth and engaging user experience across all devices.

🚀 Features :
- 🎲 Random number generation within a set range
- ⌨️ User input for guessing the number
- 💡 Hint system (“too high” / “too low”)
- 🔢 Attempt counter
- 🎉 Success message on correct guess
- 🔄 Restart game button
- 📱 Fully responsive design
- 🎨 Clean and visually appealing UI
  
🎮 How It Works :
- The system generates a random number within a defined range
- The user enters a guess
- The app provides feedback:
- Too high 📉
- Too low 📈
- Attempts are tracked automatically
- When correct, a success message is displayed
- User can restart the game anytime

🎨 UI/UX Highlights :
- Simple and intuitive layout
- Smooth transitions and feedback messages
- Centered game interface for focus
- Attractive button and input styling
- Mobile-friendly responsive design
